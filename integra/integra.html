Tab

Note the .nav-tabs class requires the .nav base class.

  • Home
  • Profile
  • Messages
<ul class="nav nav-tabs">
  <li role="presentation" class="active"><a href="#">Home</a></li>
  <li role="presentation"><a href="#">Profile</a></li>
  <li role="presentation"><a href="#">Messages</a></li>
</ul>

Tab Justified

Easily make tabs or pills equal widths of their parent at screens wider than 768px with .nav-justified. On smaller screens, the nav links are stacked.

  • Home
  • Profile
  • Messages
<ul class="nav nav-tabs nav-justified">
  ...
</ul>
<ul class="nav nav-pills nav-justified">
  ...
</ul>

Toggle

Add data-addui="toggle" to activate toggling on a single button.

<input type='toggle' data-addui="toggle">

Toggle Small

Add data-addui="toggle" to activate toggling on a single button.

Add class="toggle-sm" to make it small size.

<input type='toggle' class="toggle-sm" data-addui="toggle">

Tooltip

Add class="white-tooltip" to change tooltip background to white.

<button type="button" class="btn btn-default white-tooltip" data-toggle="tooltip" data-placement="left" title="Tooltip on left">Tooltip on left</button>

<button type="button" class="btn btn-default white-tooltip" data-toggle="tooltip" data-placement="top" title="Tooltip on top">Tooltip on top</button>

<button type="button" class="btn btn-default white-tooltip" data-toggle="tooltip" data-placement="bottom" title="Tooltip on bottom">Tooltip on bottom</button>

<button type="button" class="btn btn-default white-tooltip" data-toggle="tooltip" data-placement="right" title="Tooltip on right">Tooltip on right</button>
